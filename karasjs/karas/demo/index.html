1. Hello World! 基本的render方法接收2个参数：图表语言和渲染对象
  2. csx csx取名借鉴自jsx，Canvas-Svg-Xml，根据根节点的不同，自动切换为canvas或svg类型，两者功能完全等价
  3. svg 上面是canvas，这是一个svg的例子
  4. 高清方案 在高清屏（手机和mac pro）中，canvas会模糊或者1px边框实现问题，请自行调整meta元素的scale缩放（默认为1），或者为canvas设置外部样式，使其宽高为属性的0.5倍（对应dpr为2的倒数）
  5. tag 目前只有2种标签：div和span，分别对应block和inline元素
  6. inline 实现一个简易布局，它是css的一个（非真）子集，因为图表实际场景非常简单，部分css功能被简化，比如inline被实现成为了inline-block
  7. flex flex基本被支持，极端情况下会和标准表现稍稍不一致，但很难遇到此场景且很容器看出表现并适配
